# Basic Chatbot — CodeAlpha Python Internship

A simple rules-based terminal chatbot that responds to pre-defined user inputs using conditional logic.

## About the Project
This project is part of the CodeAlpha Python Programming Internship. The goal was to create a basic conversational agent that handles user string inputs, sanitizes them, and provides appropriate text responses.

## Features
* Standardizes user input automatically by removing trailing spaces and converting text to lowercase.
* Implements a continuous while loop for seamless interaction until the user exits.
* Handles unrecognized prompts with a default fallback message.

## Concepts Used
* While Loops & Conditional Statements (if-elif-else)
* String Methods (`strip()` and `lower()`)
